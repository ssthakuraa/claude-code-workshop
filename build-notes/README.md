# HR App Build Notes — Lab Exercise Source Material

This folder captures real experiences from building the HR Enterprise Platform with Claude Code.
Each note becomes source material for trimmed lab exercises in the training.

## How to Use

As we build each module of the HR app, we document:
1. **What technique was used** (CLAUDE.md, Skill, Plan Mode, Hook, etc.)
2. **What prompt worked** (becomes a lab instruction)
3. **What went wrong first** (becomes a "common pitfall" teaching moment)
4. **What CLAUDE.md/skill update was made** (demonstrates compounding engineering)
